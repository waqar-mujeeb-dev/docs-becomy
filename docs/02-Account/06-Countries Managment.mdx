# Adding Countries

The **`Adding Countries`** section within the **`Settings`** tab of your Becomy dashboard allows you to expand your store's reach by adding multiple countries where you can sell your products. This feature enables you to manage international sales effectively.

## Create a New Country

<img className="docs-ss" src={useBaseUrl('/img/becomy-country.png')} />

To add a new country, follow these steps:

1. Click on the **`create new country`** button.

2. Fill out the required fields in the form:
   - **Name**: Enter the name of the country (e.g., *United States*).
   - **Code**: Provide the country code (e.g., *US*).
   - **Tax Name**: Specify the name of the tax applicable to the country (e.g., *Sales Tax*).
   - **Tax**: Enter the tax percentage (e.g., *7.25* for 7.25%).
   - **Store**: Choose the store to which this country will be associated.

   <img className="docs-ss" src={useBaseUrl('/img/becomy-add-country.png')} />

3. After filling out the form, you have two options:
   - **Save**: Click this button to save the new country to your account.
   - **Cancel**: Click this button to discard the changes and return to the previous screen.



:::note

- Ensure that you enter the correct country name and code to avoid any confusion.
- The tax name and percentage should align with the regulations in the country you are adding.
- You can repeat this process to add multiple countries as needed.
:::
