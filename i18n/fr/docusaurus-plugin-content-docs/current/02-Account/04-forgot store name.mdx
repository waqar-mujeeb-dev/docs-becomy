Si vous oubliez le nom de votre magasin lors de la connexion à votre compte Becomy, vous pouvez facilement retrouver le nom de votre magasin.

### Connaitre votre nom de magasin

1. Depuis la page de connexion Becomy, cliquez sur `oublié le nom du magasin`.
2. Entrez l'email associé à votre compte Becomy.
3. Cliquez sur `Obtenir les magasins`.
4. Vous recevrez un email avec les noms de tous vos `noms de magasins`.
