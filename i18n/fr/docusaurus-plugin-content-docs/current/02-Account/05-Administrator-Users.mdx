# Administrateurs

La section **`Administrateurs`** dans l'onglet **`Paramètres`** vous permet de gérer les utilisateurs administrateurs de votre boutique Becomy. Vous pouvez **`ajouter`**, **`modifier`** ou **`supprimer`** des utilisateurs administrateurs selon vos besoins.

## Gestion des Administrateurs

<img className="docs-ss" src={useBaseUrl('/img/becomy-administrator.png')} />

Dans cette section, vous avez plusieurs options pour gérer les utilisateurs administrateurs :

- **Voir les Administrateurs** : Voir la liste de tous les utilisateurs administrateurs associés à votre boutique. 

- **Modifier un Administrateur** : Modifier les informations d'un utilisateur administrateur existant.

- **Supprimer un Administrateur** : Retirer un utilisateur administrateur de votre boutique si son accès n'est plus nécessaire.

- **Ajouter un Nouvel Administrateur** : Créer un nouvel utilisateur administrateur en fournissant les détails requis, tels que le `Prénom`, le `Nom de famille`, l'`adresse e-mail` et le `mot de passe`.

## Ajouter un Nouvel Administrateur

Pour ajouter un nouvel administrateur, suivez ces étapes :

1. Cliquez sur le bouton **`créer un nouvel administrateur`**.
2. Remplissez le formulaire avec les informations nécessaires, y compris le `Prénom`, le `Nom de famille`, l'`adresse e-mail` et le `mot de passe`.
3. Cliquez sur **`Enregistrer`** pour ajouter le nouvel utilisateur à la liste des administrateurs.
