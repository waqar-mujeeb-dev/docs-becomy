# Ajout de Pays

La section **`Ajout de Pays`** dans l'onglet **`Paramètres`** de votre tableau de bord Becomy vous permet d'élargir la portée de votre boutique en ajoutant plusieurs pays où vous pouvez vendre vos produits. Cette fonctionnalité vous permet de gérer efficacement les ventes internationales.

## Créer un Nouveau Pays

<img className="docs-ss" src={useBaseUrl('/img/becomy-country.png')} />

Pour ajouter un nouveau pays, suivez ces étapes :

1. Cliquez sur le bouton **`créer un nouveau pays`**.

2. Remplissez les champs requis dans le formulaire :
   - **Nom** : Entrez le nom du pays (ex. : *États-Unis*).
   - **Code** : Fournissez le code du pays (ex. : *US*).
   - **Nom de la Taxe** : Spécifiez le nom de la taxe applicable au pays (ex. : *Taxe de Vente*).
   - **Taxe** : Entrez le pourcentage de la taxe (ex. : *7,25* pour 7,25 %).
   - **Boutique** : Choisissez la boutique à laquelle ce pays sera associé.

   <img className="docs-ss" src={useBaseUrl('/img/becomy-add-country.png')} />

3. Après avoir rempli le formulaire, vous avez deux options :
   - **Enregistrer** : Cliquez sur ce bouton pour enregistrer le nouveau pays dans votre compte.
   - **Annuler** : Cliquez sur ce bouton pour annuler les modifications et revenir à l'écran précédent.

:::note

- Assurez-vous d'entrer le nom et le code du pays correctement pour éviter toute confusion.
- Le nom et le pourcentage de la taxe doivent être conformes aux réglementations du pays que vous ajoutez.
- Vous pouvez répéter ce processus pour ajouter plusieurs pays selon vos besoins.
:::
