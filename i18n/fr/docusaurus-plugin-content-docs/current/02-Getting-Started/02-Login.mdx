D'accord, vous avez déjà configuré votre compte – high five ! Maintenant, chaque fois que vous souhaitez replonger dans votre monde Becomy, suivez ces étapes :

1. Ouvrez votre navigateur web et allez sur [Becomy](https://MyShop.becomystaging.com).

:::info

Avant de cliquer sur Connexion, n'oubliez pas d'ajouter votre nom de magasin unique comme préfixe à l'URL. Par exemple, si le nom de votre magasin est "MyShop", votre URL de connexion serait `https://MyShop.becomystaging.com`.

:::

2. Recherchez le bouton `Connexion`, généralement en haut à droite de la page.
3. Tapez l'adresse e-mail que vous avez utilisée lors de l'inscription.
4. Vous vous souvenez de ce mot de passe que vous avez choisi lors de l'inscription ? Il est temps de le mettre à profit.
5. Cliquez sur le bouton "Connexion", et vous êtes de retour en action !
6. Si jamais vous oubliez votre mot de passe, pas de soucis. Il vous suffit de cliquer sur le lien "Mot de passe oublié", et nous vous guiderons à travers une réinitialisation rapide.

Vous pouvez également vous connecter rapidement en utilisant votre `compte Google`. Il vous suffit de cliquer sur le bouton `Se connecter avec Google` sur la page de connexion et de suivre les étapes simples pour lier vos identifiants Google.

:::info

N'oubliez pas de garder vos **_identifiants de connexion_** en sécurité. Votre parcours Becomy est personnel, alors gardons-le ainsi.

:::
