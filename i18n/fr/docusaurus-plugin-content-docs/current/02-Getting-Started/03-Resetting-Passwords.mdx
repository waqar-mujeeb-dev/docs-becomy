Si vous oubliez votre mot de passe de connexion ou si vous souhaitez améliorer la sécurité de votre compte, vous pouvez facilement le réinitialiser. En tant que propriétaire du magasin, vous pouvez réinitialiser votre propre mot de passe et également initier des réinitialisations de mot de passe pour vos employés.

Le processus de réinitialisation d'un mot de passe reste le même, même avec l'authentification à deux étapes activée.

### Réinitialiser votre mot de passe oublié

1. Depuis la page de connexion Becomy, cliquez sur `mot de passe oublié`.

<img className="docs-ss" src={useBaseUrl('/img/login-forgot-password-becomy.png')} />

2. Entrez le nom de domaine de la boutique et l'adresse e-mail associée à votre compte Becomy.

<img className="docs-ss" src={useBaseUrl('/img/admin-reset-password-becomy.png')} />

3. Cliquez sur **Réinitialiser le mot de passe**.
4. Vous recevrez un email avec un lien pour réinitialiser votre mot de passe. Cliquez sur `Réinitialiser le mot de passe` dans cet email.
5. Entrez votre nouveau mot de passe dans le champ `confirmer le mot de passe`.
6. Cliquez sur `Réinitialiser le mot de passe`.
7. Retournez à la page de connexion Becomy et connectez-vous avec votre nouveau mot de passe.
