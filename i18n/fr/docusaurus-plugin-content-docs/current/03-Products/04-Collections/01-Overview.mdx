Les collections aident à organiser les produits en catégories, facilitant ainsi la recherche pour les clients. Les collections peuvent regrouper des articles similaires, comme **Robes d'été** ou **Électronique**. Cela améliore l'expérience d'achat, aide les clients à découvrir plus de produits et peut augmenter les ventes.

Les collections peuvent être affichées sur la page produit pour aider les utilisateurs à naviguer et explorer les produits associés. Lorsqu'un produit fait partie d'une collection, le nom de la collection est souvent affiché sur la page produit. Les utilisateurs peuvent cliquer sur le nom de la collection pour voir la page dédiée à la collection, qui présente des images et des produits de cette collection. 
Cela permet aux clients de voir tous les articles regroupés sous la même catégorie, facilitant ainsi la découverte de produits supplémentaires qui pourraient les intéresser. Le design et la mise en page des pages de collection sont déterminés par votre thème.

Pour gérer vos collections, accédez à la section **Collections** dans votre admin Becomy. Ici, vous pouvez voir les collections existantes, en créer de nouvelles et apporter des modifications si nécessaire.

## Types de Collections

Vous pouvez créer deux types de collections : automatisées et manuelles. 

### Collections Automatisées

Les collections automatisées utilisent des conditions pour inclure automatiquement les produits correspondants. Par exemple, vous pouvez définir une condition pour inclure tous les produits étiquetés "Été" dans une **Collection d'Été**. Ce type de collection se met à jour automatiquement lorsque vous ajoutez de nouveaux produits répondant aux conditions.

### Collections Manuelles

Les collections manuelles nécessitent d'ajouter individuellement chaque produit. Cela vous donne un contrôle total sur les produits inclus. Par exemple, vous pouvez créer une collection **Produits Vedettes** et sélectionner manuellement les articles que vous souhaitez mettre en avant.

### Affichage des Collections

Une fois créées, les collections peuvent être affichées sur votre boutique en ligne sous forme de page web avec une galerie de produits. Les clients peuvent cliquer sur une image de produit pour visiter la page spécifique du produit. Pour aider les clients à trouver les collections, vous pouvez ajouter des liens vers celles-ci dans le menu de navigation de votre boutique.

La mise en page et l'apparence exactes des pages de collection dépendent de votre thème. 

Vous pouvez voir, créer et modifier des collections depuis la page **Collections** dans votre admin Becomy.
