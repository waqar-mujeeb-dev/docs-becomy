# Collections Manuelles

Une collection manuelle sur Becomy vous permet de créer un ensemble spécifique de produits. Ces collections restent statiques à moins que vous n'ajoutiez ou supprimiez des produits manuellement. 
Ce guide vous expliquera les étapes pour créer, gérer et supprimer des collections manuelles.
Les collections manuelles sont idéales pour regrouper des produits que vous souhaitez mettre en avant ensemble. Contrairement aux collections automatiques, les collections manuelles nécessitent que vous sélectionniez chaque produit individuellement.

:::info
Vous ne pouvez pas convertir une collection automatique en collection manuelle. Si vous avez besoin d'une collection manuelle, vous devez la créer de toutes pièces.
:::

## Créer une Collection Manuelle
Suivez ces étapes pour créer une collection manuelle :

1. Allez dans **Produits > Collections** dans votre admin Becomy.
2. Cliquez sur **Créer une Collection**.
3. Entrez un titre et une description pour votre collection.
4. Dans la section **Type de Collection**, sélectionnez **Manuelle**.
5. Cliquez sur **Enregistrer**.
6. Dans la section **Produits**, recherchez des produits ou cliquez sur **Parcourir** pour ajouter des produits à la collection.
7. Définissez l'ordre de tri des produits.
8. Facultativement, modifiez l'aperçu de la liste des moteurs de recherche.
9. Sélectionnez les canaux de vente où la collection sera disponible.
10. Téléchargez une image pour la collection si désiré.
11. Cliquez sur **Enregistrer**.

## Ajouter des Produits à une Collection Manuelle
Pour ajouter des produits à une collection manuelle existante :

1. Allez dans **Produits > Collections**.
2. Cliquez sur le nom de la collection.
3. Dans la section **Produits**, recherchez des produits ou cliquez sur **Ajouter un Produit** pour les ajouter.

## Supprimer des Produits d'une Collection Manuelle
Pour supprimer des produits d'une collection manuelle :

1. Allez dans **Produits > Collections**.
2. Cliquez sur le nom de la collection.
3. Dans la section **Produits**, cliquez sur le **x** à côté du produit que vous souhaitez supprimer.

## Supprimer une Collection
Pour supprimer une collection manuelle :

1. Allez dans **Produits > Collections**.
2. Cliquez sur le nom de la collection que vous souhaitez supprimer.
3. Cliquez sur **Supprimer la Collection**.
4. Confirmez la suppression dans la boîte de confirmation.

:::info
Les collections supprimées sont définitivement supprimées. Les liens vers les collections supprimées sont automatiquement retirés des menus de votre boutique en ligne.
:::
En suivant ces étapes, vous pouvez gérer efficacement vos collections manuelles sur Becomy, en veillant à ce que vos produits soient organisés et facilement accessibles à vos clients.
