## Utilisation de l'éditeur de texte enrichi
En utilisant cet éditeur de texte enrichi intégré, vous pouvez créer la description de votre produit, des informations sur la catégorie, des instructions de paiement ou d'expédition, et bien plus encore. Vous pourrez mettre en valeur les caractéristiques de votre produit et fournir un moyen clair et concis pour que les clients comprennent votre produit.

Cet éditeur vous permet de formater du texte, d'ajouter des liens, des images et des vidéos, de modifier la taille, la couleur et la police du texte. De plus, vous pouvez ajouter du balisage HTML à votre contenu, ce qui vous permet de créer des styles et des effets personnalisés.
Ces détails vous aideront à présenter votre produit de manière plus attrayante.

### Formater du texte avec l'éditeur de texte enrichi

Le formatage du texte vous aidera à mieux structurer votre contenu et le rendre plus lisible pour les personnes et les moteurs de recherche.

Il existe plusieurs options de formatage disponibles dans cet éditeur de texte enrichi :
- Texte normal (Arial 12px)
- Titre 1
- Titre 2
- Titre 3
- Titre 4
- Titre 5
- Titre 6

Pour modifier le formatage d'un texte, surlignez le texte correspondant et choisissez les options de formatage pertinentes.

### Polices de caractères

#### Souligner du texte
Utilisez le bouton Souligner pour mettre le texte en souligné.

#### Mettre du texte en italique
Pour mettre du texte en italique, utilisez le bouton Italique.

#### Mettre du texte en gras
Pour mettre du texte en gras, utilisez le bouton Gras.

#### Barrer du texte
Pour barrer ou supprimer du texte, sélectionnez le texte que vous souhaitez barrer et cliquez sur le bouton Barré dans la barre d'outils de l'éditeur de texte enrichi.

### Ajouter des Liens

Vous pouvez ajouter des hyperliens dans les descriptions de produits et de collections avec cet éditeur de texte enrichi intégré. De plus, vous pouvez ajouter des liens qui dirigeront les clients vers n'importe quelle page de votre magasin en ligne sur Becomy.
De plus, vous pouvez ajouter des liens qui permettront à vos clients de vous appeler ou de vous envoyer un email.

1. Mettez en surbrillance le texte que vous souhaitez transformer en lien hypertexte.
2. Cliquez sur l'icône de lien dans la barre d'outils de l'éditeur de texte enrichi.
3. Dans la boîte de dialogue qui apparaît, saisissez l'URL vers laquelle vous souhaitez créer un lien.
   - Pour créer un lien vers un site externe, vous pouvez écrire _`https://example.com`_.
   - Pour ajouter un lien vers une page de votre magasin en ligne, vous pouvez écrire _`/collections/all`_.
   - Pour permettre à vos clients de vous appeler, vous pouvez écrire _`tel:+1234567890`_.
   - Pour vous envoyer un email, vous pouvez écrire _`mailto:support@example.com`_.
4. Facultativement, vous pouvez choisir d'ouvrir le lien dans un nouvel onglet ou dans l'onglet existant en cochant la case correspondante.
5. Cliquez sur le bouton "Insérer" pour ajouter le lien hypertexte.

### Ajouter des Images
Vous pouvez ajouter des images depuis votre système ou à partir de certaines URL externes. Veuillez vous assurer de sélectionner le format d'image autorisé pris en charge par Becomy, c'est-à-dire PNG, JPEG, JPG et WebP.
La taille maximale pour télécharger une image est de 8 Mo.

Pour insérer une image, cliquez sur l'icône d'image dans la barre d'outils de l'éditeur de texte enrichi.

<img className="docs-ss" src={useBaseUrl('/img/rte-add-image.png')} />

Une fois que vous avez cliqué sur l'icône, vous verrez une fenêtre contextuelle **pour Insérer/Modifier l'image**,

<img className="docs-ss w-33"  src={useBaseUrl('/img/rte-edit-image.png')} />
1. Vous pouvez sélectionner l'image depuis votre **système ou à partir d'une URL**.
2. Facultativement, vous pouvez ajouter un **titre d'image**, une **description** et un **texte alternatif**.
3. Vous pouvez choisir de maintenir le rapport largeur-hauteur original d'une image lors de sa redimensionnement (ce qui est sélectionné via l'icône de verrouillage).
4. Lorsque vous téléchargez une image, les dimensions de l'image seront automatiquement récupérées et affichées dans les champs correspondants de hauteur et de largeur.
5. Vous pouvez également définir manuellement les dimensions dans les champs _hauteur_ et _largeur_.
6. Enfin, cliquez sur Enregistrer pour insérer l'image.

:::info
Ajouter une image dans l'éditeur de texte enrichi n'est pas un processus de glisser-déposer. Vous devez ajouter le chemin de l'image.
:::

### Ajouter des vidéos
Vous pouvez afficher votre produit en ajoutant une URL vidéo. Assurez-vous simplement que votre contenu est utile pour les clients.

    <img className="docs-ss" src={useBaseUrl('/img/rte-insert-vid-btn.png')} />

Pour ajouter une vidéo à votre boîte de description :

1. Cliquez sur l'icône vidéo située dans la barre d'outils de l'éditeur de texte enrichi pour lancer le processus d'insertion de la vidéo.
2. Une boîte de dialogue apparaîtra, vous présentant deux options distinctes pour ajouter votre contenu vidéo :
    - **Général** : Utilisez cette option pour ajouter directement une vidéo via une URL. Collez simplement le lien de votre vidéo dans le champ prévu à cet effet. C'est idéal pour partager rapidement des vidéos hébergées sur des plateformes comme YouTube ou Vimeo.

        <img className="docs-ss" src={useBaseUrl('/img/rte-insert-media-general.png')} />
    - **Intégrer** : Choisissez cette option si vous avez un code d'intégration pour votre vidéo. Les codes d'intégration sont généralement fournis par les services d'hébergement vidéo et permettent une personnalisation plus poussée de la manière dont la vidéo est affichée sur votre site.

				<img className="docs-ss" src={useBaseUrl('/img/rte-insert-media-embed.png')} />

3. Après avoir sélectionné votre méthode préférée et configuré vos paramètres vidéo, cliquez sur le bouton "Insérer" pour intégrer la vidéo dans votre contenu texte enrichi.

### Créer des listes
Cet éditeur de texte enrichi vous permet de créer des listes ordonnées et non ordonnées. La première utilise des nombres et la seconde des puces.

- Pour créer une **liste non ordonnée**, cliquez sur le bouton de liste à puces. Alternativement, vous pouvez commencer en rendant le premier élément de la liste en puces, puis appuyez sur la touche entrée pour créer un nouvel élément de liste. Pour terminer la liste, appuyez deux fois sur la touche entrée.

<img className="docs-ss" src={useBaseUrl('/img/rte-unordered-list.png')} />
- Pour créer une **liste ordonnée**, cliquez sur le bouton de liste numérotée. De même, vous pouvez commencer en rendant le premier élément de la liste numéroté, puis appuyez sur la touche entrée pour créer un nouvel élément de liste. Pour terminer la liste, appuyez deux fois sur la touche entrée.

<img className="docs-ss" src={useBaseUrl('/img/rte-ordered-list.png')} />

### Alignement du texte dans l'éditeur de texte enrichi

Il existe quatre types d'alignements disponibles dans l'éditeur de texte enrichi, qui sont les suivants :

- Aligner le texte à gauche
- Centrer le texte
- Aligner le texte à droite
- Justifier le texte

<img className="docs-ss" src={useBaseUrl('/img/rte-text-alignment.png')} />

Ces options d'alignement vous aident à contrôler la mise en page et la présentation de votre contenu de manière efficace.

### Appliquer des indentations

Les indentations donneront à vos paragraphes une marge du côté gauche, les mettant en valeur. Pour indenter ou désindenter du texte dans l'éditeur de texte enrichi, suivez ces étapes :

- **Indenter le texte :** Sélectionnez le texte que vous souhaitez indenter et cliquez sur le bouton d'indentation dans la barre d'outils de l'éditeur de texte enrichi.
- **Désindenter le texte :** Sélectionnez le texte indenté que vous souhaitez désindenter et cliquez sur le bouton de désindentation dans la barre d'outils de l'éditeur de texte enrichi.

L'utilisation des boutons d'indentation et de désindentation vous aidera à ajuster l'alignement de votre texte de manière efficace.

### Personnaliser le style de police avec l'éditeur de texte enrichi

Pour personnaliser le style de police dans l'éditeur de texte enrichi, suivez ces étapes :

- **Couleur de la police :** Cliquez sur le bouton de couleur de police pour changer la couleur du texte.
- **Taille de la police :** Utilisez le menu déroulant de taille de police pour sélectionner la taille de police souhaitée pour le texte.

<img className="docs-ss" src={useBaseUrl('/img/rte-font-size.png')} />

- **Famille de police :** Cliquez sur le menu déroulant de famille de police pour choisir la famille de police souhaitée pour le texte.

<img className="docs-ss" src={useBaseUrl('/img/rte-font-family.png')} />

- **Couleur de fond :** Pour changer la couleur de fond du texte, mettez en surbrillance le texte et cliquez sur le bouton de couleur de fond dans la barre d'outils de l'éditeur de texte enrichi.

<img className="docs-ss" src={useBaseUrl('/img/rte-bg-color.png')} />

Il existe plusieurs façons de personnaliser les paramètres de police dans un éditeur de texte.

Une méthode courante consiste à sélectionner un morceau de texte spécifique, puis à utiliser le bouton de paramètres de police correspondant dans le menu de l'éditeur de texte. En faisant ainsi, le style de police pour cette sélection de texte particulière sera modifié en fonction du paramètre choisi.

Une autre façon de personnaliser les paramètres de police est de presser directement le bouton de paramètres de police avant de taper. Une fois le bouton activé, tout texte tapé par la suite adoptera automatiquement le style de police sélectionné. Pour passer à un autre paramètre de police, appuyez simplement à nouveau sur le bouton pour choisir un nouveau style.

:::tip
Pour les couleurs qui ne sont pas disponibles dans la palette standard, les utilisateurs peuvent éditer leur texte en mode HTML pour appliquer des couleurs personnalisées à leur contenu.
:::
