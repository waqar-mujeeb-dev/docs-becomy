# Traitement des Commandes

Lorsqu'un client achète dans votre boutique en ligne ou tout autre canal de vente, cela apparaît comme une nouvelle commande sur la page des Commandes dans Becomy.

## Que se passe-t-il lorsque vous recevez une commande en ligne

- Vous recevez une notification de nouvelle commande par e-mail.
- La page des Commandes dans Becomy affiche la nouvelle commande.
- Le client reçoit un e-mail de confirmation de commande.
- Le traitement de la commande commence.

:::info

Un checkout abandonné est créé si une commande n'a pas été finalisée, mais que le client a atteint le checkout.
:::

## Flux de traitement typique des commandes en ligne

Chaque commande en ligne passe par 3 étapes. À chaque étape, une action différente est requise dans Becomy. Vous pouvez soit configurer Becomy pour effectuer ces actions automatiquement, soit les gérer manuellement :
