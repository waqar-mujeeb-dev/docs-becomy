# Managing Pages on Becomy

Becomy vous permet de créer, personnaliser et gérer facilement les pages de votre boutique grâce à son tableau de bord d'administration convivial.

<img className="docs-ss" src={useBaseUrl('/img/pages-becomy.png')} />

## Setting Up Your Page

Pour commencer à créer une page, accédez à votre **`Tableau de bord d'administration Becomy`** et ouvrez l'onglet **`Modèle de boutique`**. Sous cet onglet, sélectionnez **`Pages`** pour accéder à la section des pages.

## Creating a New Page

- **Créer une nouvelle page**: Dans la section Pages, cliquez sur le bouton "`Créer une nouvelle page`" situé dans le coin supérieur droit. Cela ouvrira un formulaire qui vous permet d'écrire, formater et modifier facilement le contenu de votre page.

---

Ce formulaire vous permet de créer une nouvelle page pour votre boutique. Remplissez les champs requis tels que:

- **Titre**: L'en-tête principal ou le nom de la page.
- **Titre de la page**: Un titre secondaire affiché dans l'onglet du navigateur.
- **Description de la page**: Un résumé ou une description du contenu de la page.
- **Catégorie**: Sélectionnez la catégorie appropriée (par exemple, page_de_destination) pour organiser votre page.

Une fois tous les champs remplis, cliquez sur le bouton "Sauvegarder" pour créer la page.

<img className="docs-ss" src={useBaseUrl('/img/create-new-page-becomy.png')} />

## Page Management Tools: Edit, Preview, and Delete

Une fois que vous avez créé la page, sa gestion est simple depuis votre tableau de bord Becomy.

- **Aperçu de la page**: La fonctionnalité d'aperçu vous permet de voir à quoi ressemblera votre page pour les visiteurs avant sa publication. Elle offre un affichage visuel précis du contenu de la page, y compris le texte, les images et la mise en page. Pour prévisualiser la page, localisez-la dans la section "`Pages`" et cliquez sur "`l'icône de l'œil`".

<img className="docs-ss" src={useBaseUrl('/img/page-preview-becomy.png')} />

- **Suppression de la page**: Si vous devez supprimer une page, localisez-la dans la section "`Pages`" et cliquez sur `l'icône de suppression`. Une invite de confirmation apparaîtra pour vous assurer que vous souhaitez continuer. `Confirmez la suppression` pour supprimer définitivement la page de votre boutique.
