# Préférences

La section **`Préférences`** dans l'onglet `Modèle de Boutique` vous permet de personnaliser les détails essentiels de votre boutique Becomy, y compris les `paramètres SEO`, l'URL et l'intégration avec les `outils d'analyse`. Cette section est conçue pour vous aider à contrôler l'apparence de votre boutique pour les clients et sur les moteurs de recherche.

## Détails de la Boutique

Cette section permet de visualiser les informations clés de votre boutique :

- **Nom** : Ceci est le `nom affiché` de votre boutique.  

- **Titre** : Un titre personnalisable pour votre boutique, souvent affiché dans l'onglet du navigateur.  

- **URL** : L'`URL unique` où les clients peuvent accéder à votre boutique.  

- **Code Pays** : Spécifie le `pays principal` associé à votre boutique.  


## Préférences de la Page d'Accueil

Cette section vous permet de contrôler l'apparence de votre boutique sur les moteurs de recherche, améliorant ainsi votre visibilité.

- **Titre SEO** : Le titre qui apparaît dans les résultats des moteurs de recherche pour la page d'accueil de votre boutique.  
  **Exemple** : *Iphone*

- **Description SEO** : Une brève description qui apparaît à côté du titre dans les résultats des moteurs de recherche.  
  **Exemple** : *Description SEO*

  <img className="docs-ss" src={useBaseUrl('/img/becomy-seo.png')} />

- **Aperçu de l'Image de la Boutique en Ligne** : Une image qui apparaît lors du partage du lien de votre boutique sur les `réseaux sociaux` ou autres plateformes.  
  - **Instructions de Téléchargement** : Vous pouvez télécharger un fichier `PNG`, `JPG` ou `GIF` jusqu'à 10 Mo, ou glisser-déposer votre image préférée.

<img className="docs-ss" src={useBaseUrl('/img/becomy-preview.png')} />

## Canaux de Vente

Vous pouvez étendre votre présence en ligne en configurant différents canaux de vente :

- **Modifier les Modifications** : Personnalisez et ajoutez des canaux de vente pour atteindre un `public plus large`.

- **Compte Google Analytics** : Connectez votre boutique à `Google Analytics` pour suivre et analyser le trafic.  

- **Pixel Facebook** : Intégrez le `Pixel Facebook` pour le suivi et l'analyse.  

- **Twitter** : Connectez votre compte `Twitter` pour le suivi.
