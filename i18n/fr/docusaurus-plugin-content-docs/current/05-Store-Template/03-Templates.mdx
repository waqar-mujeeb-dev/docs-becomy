# Design

La section **`Design`** dans l'onglet **`Modèle de Boutique`** permet aux utilisateurs de choisir et personnaliser le design de leur boutique Becomy. Similaire à Shopify, cette section offre diverses `options de modèles` et une flexibilité de design pour aider les utilisateurs à créer une boutique visuellement attrayante.

## Boutique de Modèles

<img className="docs-ss" src={useBaseUrl('/img/becomy-template.png')} />

Cette page permet aux utilisateurs de parcourir et sélectionner parmi une variété de modèles pour le design de leur boutique. Les utilisateurs peuvent également télécharger leurs propres modèles s'ils préfèrent un design personnalisé.

- **Design Actuel** : Affiche le modèle actuellement sélectionné pour la boutique.  
  **Exemple**: *Becomy*

- **Parcourir les Modèles** : Permet aux utilisateurs d'explorer les modèles disponibles dans la bibliothèque pour trouver celui qui correspond au style et à l'image de marque de leur boutique.

<img className="docs-ss" src={useBaseUrl('/img/becomy-template-browse.png')} />

- **Modèles Enregistrés** : Affiche les modèles que l'utilisateur a `enregistrés` pour un accès facile et une utilisation future.  

  <img className="docs-ss" src={useBaseUrl('/img/becomy-save-template.png')} />

## Gestion des Modèles

- **Télécharger des Modèles** : Les utilisateurs ont la possibilité de `télécharger des modèles personnalisés` s'ils ont un design spécifique qu'ils souhaitent utiliser pour leur boutique.

- **Aperçu** : Une fois un modèle sélectionné, les utilisateurs peuvent `prévisualiser` l'apparence de leur boutique avec le design choisi.
