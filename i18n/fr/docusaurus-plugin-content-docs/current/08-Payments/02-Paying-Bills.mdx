# Paiement des factures sur Becomy avec une carte de crédit

Vous pouvez gérer différents types de factures directement via votre admin Becomy, toutes payables avec votre `carte de crédit` :

- **Paiements de factures générales** : Pour les factures générales de Becomy, comme le paiement de votre abonnement, vous pouvez utiliser une `carte de crédit` comme méthode de paiement.

- **Facturation des frais d'expédition** : Les frais et ajustements des étiquettes d'expédition sont facturés sur la `carte de crédit` associée à votre `compte Becomy`. Tous les frais d'expédition sont facturés dans la devise applicable en fonction des services d'expédition utilisés.

## Payer vos factures Becomy

Toutes vos factures Becomy, y compris les paiements généraux et les frais d'expédition, seront payées avec votre carte de crédit. Assurez-vous de vérifier vos informations de facturation pour que vos paiements soient traités sans problème.

## Gérer vos méthodes de paiement

Vous pouvez ajouter et gérer vos méthodes de paiement via votre tableau de bord Becomy :

1. Accédez à votre **tableau de bord Becomy**.
2. Dans la barre latérale, cliquez sur l'onglet **Paramètres**.
3. Dans le menu déroulant des paramètres, sélectionnez **Informations de facturation**.
4. Ici, vous pouvez ajouter, mettre à jour ou gérer vos `options de paiement`.

Assurez-vous que vos `informations de facturation` sont à jour pour un traitement fluide des paiements.
